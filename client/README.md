# Проект: Веб-платформа Книжного клуба
**Автор:** Адия
**Стек:** React, Node.js, MongoDB.

## Как запустить:
1. В терминале `server`: `npm install` -> `node server.js`
2. В терминале `client`: `npm install` -> `npm start`

## Функционал:
- Просмотр каталога книг (React + Axios)
- Серверная часть на Express
- Облачное хранение данных в MongoDB Atlas
